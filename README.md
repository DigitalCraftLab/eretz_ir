# ארץ עיר

אפליקציית MVP למשחק `ארץ עיר` בעברית, עם חדרים משותפים ל-2 עד 6 שחקנים, כל אחד מהמכשיר שלו.

## מה יש בגרסה הזו

- יצירת חדר והצטרפות עם קוד חדר
- בחירת 4 קטגוריות קבועות לכל המשחק
- בחירת משך סבב: `1` דקה, `1.5` דקות או `3` דקות
- הצעת קטגוריות ידנית או קבלת הצעות אקראיות
- 4 סבבים עם אות עברית אקראית חדשה בכל סבב
- שמירה אוטומטית של התשובות בזמן ההקלדה
- בדיקה לפי קטגוריה בסוף כל סבב
- ניקוד:
  - `10` לתשובה מאושרת וייחודית
  - `5` לתשובה מאושרת שזהה או דומה מאוד לתשובה של משתתף אחר
  - `0` לחסר, לתשובה שלא מתחילה באות, או לתשובה שלא אושרה
  - `+1` לכל לייק שמשתתפים אחרים נותנים בזמן הבדיקה

## הרצה

```bash
python3 server.py
```

אחר כך פותחים דפדפן בכתובת:

```text
http://localhost:8000
```

כדי לשחק מכמה מכשירים באותה רשת, פותחים את השרת מהמחשב המארח ונכנסים מהטלפונים או המחשבים האחרים לכתובת ה-LAN של המחשב, למשל:

```text
http://192.168.1.15:8000
```

## פריסה ל-Render

1. דוחפים את הפרויקט ל-GitHub.
2. נכנסים ל-[Render](https://render.com) ויוצרים `Blueprint` חדש מהמאגר.
3. Render יקרא את הקובץ `render.yaml` ויצור שירות web.
4. אחרי שהפריסה מסתיימת, תקבלו URL ציבורי שאפשר לשתף עם שחקנים.

אם מעדיפים ליצור את השירות ידנית במקום `Blueprint`, משתמשים בהגדרות האלה:

- `Environment`: Python 3
- `Build Command`: `pip install -r requirements.txt`
- `Start Command`: `python3 server.py`

הערה חשובה:

- כרגע כל מצב המשחק נשמר בזיכרון השרת. אם Render יעשה restart לשירות, החדרים הפעילים יימחקו.

## הערות MVP

- כל המידע נשמר כרגע בזיכרון השרת, אז ריענון השרת מוחק חדרים פעילים.
- "דומה מאוד" ממומש כרגע באמצעות נרמול פשוט של כתיב עברי נפוץ, רווחים, מקפים ואותיות סופיות.
- המארח הוא זה שבוחר את ארבע הקטגוריות, מעביר בין קטגוריות הבדיקה, ומאשר אם מילה קיימת ומתאימה לקטגוריה.
